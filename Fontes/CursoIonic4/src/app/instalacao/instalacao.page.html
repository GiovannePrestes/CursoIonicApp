Instalando ionic via node

Abra o prompt de comando de preferencia no modo administrador e execute o comando abaixo
npm install -g ionic

Criando o Projeto

Para criar o projeto é bem simples, vamos utilizar o comando abaixo:
ionic start NomeDoSeuApp tabs --type=angular

Logo em seguida digite o seguinte comando no prompt :
cd NomeDoSeuApp
ionic cordova platform
code .
Code irá abrir o visual studio code na pasta do projeto

Rodando o projeto

  • Para rodar seu projeto, digite no prompt de comando:

    • ionic serve

  • Rodar projeto direto no celular:

    • ionic cordova run android

    Gerando APK

    ionic cordova build android
